# Python Essentials

## Basic Syntax

Variables and types:

```python
name = "John"
age = 30
is_active = True
items = [1, 2, 3]
data = {"key": "value"}
```

## Functions

Define functions:

```python
def greet(name, greeting="Hello"):
    return f"{greeting}, {name}!"

def calculate(a, b, *args, **kwargs):
    return sum([a, b] + list(args))

lambda x: x * 2
```

## Lists and Comprehensions

List operations:

```python
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
numbers.extend([7, 8])
numbers[0]
numbers[-1]
numbers[1:3]

squared = [x**2 for x in numbers]
evens = [x for x in numbers if x % 2 == 0]
```

## Dictionaries

Dictionary operations:

```python
user = {"name": "John", "age": 30}
user["email"] = "john@example.com"
user.get("phone", "N/A")
user.keys()
user.values()
user.items()

{k: v for k, v in user.items() if v}
```

## Control Flow

Conditionals and loops:

```python
if condition:
    pass
elif other_condition:
    pass
else:
    pass

for item in items:
    print(item)

for i, item in enumerate(items):
    print(i, item)

while condition:
    break
    continue
```

## Classes

Object-oriented programming:

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        return f"Hello, I'm {self.name}"

    @property
    def is_adult(self):
        return self.age >= 18

person = Person("John", 30)
```

## File Operations

Read and write files:

```python
with open("file.txt", "r") as f:
    content = f.read()
    lines = f.readlines()

with open("output.txt", "w") as f:
    f.write("Hello\n")
    f.writelines(["line1\n", "line2\n"])
```

## Error Handling

Try-except blocks:

```python
try:
    result = 10 / 0
except ZeroDivisionError as e:
    print(f"Error: {e}")
except Exception as e:
    print(f"Unexpected: {e}")
finally:
    print("Cleanup")
```

## Common Patterns

Useful patterns:

```python
# Enumerate
for i, item in enumerate(items):
    pass

# Zip
for a, b in zip(list1, list2):
    pass

# Map and filter
result = list(map(lambda x: x*2, items))
filtered = list(filter(lambda x: x > 0, items))

# Any and all
any(x > 10 for x in numbers)
all(x > 0 for x in numbers)
```
