Design guidelines :
Design is always mobile first
Never put border boxes inside other border boxes.
Use no more thant 4 shades of gray
Use black and white
Be minimalist
Use a single font
use typescales
eveything should be based on baseline grid
Always use design tokens and var
Dont use pixel
Use modern css properties like : layers, import, clamp, has:

Avoid at all cost :
- rounded corners
- shadows
- colors
- purple
- gradients
- icons
- svgs
- lorem ipsum
